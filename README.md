# Prompt Vault — AI提示词宝库

[![Live Demo](https://img.shields.io/badge/Live-prompt--vault.vercel.app-f97316)](https://prompt-vault.vercel.app)

A searchable, categorized library of ready-to-use AI prompts for ChatGPT, Claude, Midjourney, and more. Bilingual (EN/CN), free, open-source.

## Features

- **30+ prompts** across 9 categories (growing)
- **Instant search** — filter by keyword in English or Chinese
- **One-click copy** — paste directly into any AI tool
- **Category sidebar** — Writing, Coding, Image, Business, Learning, Productivity, Data, Creative, AI
- **Tool tags** — see which prompts work with ChatGPT, Claude, Midjourney, etc.
- **Bilingual** — every prompt has a Chinese description

## Categories

| Category | Prompts | Examples |
|----------|---------|---------|
| Writing | Blog posts, emails, LinkedIn, resumes, Twitter threads | |
| Coding | Code review, debugging, unit tests, SQL, language conversion | |
| Image | Midjourney portraits, logos, illustrations, product photos | |
| Business | SWOT analysis, pitch decks, cold emails | |
| Learning | Feynman technique, flashcards, book summaries | |
| Productivity | Meeting notes, weekly planning, decision matrices | |

## Tech

Single-page HTML/CSS/JS with client-side search. No framework, no build step.
